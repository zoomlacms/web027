美人椒的前世今生

400年前漂洋过海到神州大地

辣椒原产地南美热带，墨西哥是辣椒的发源地，全球约一半辣椒都生长于墨西哥境内，辣椒被墨西哥视为“国食”。辣椒于明代传入我国，《遵生八笺》其中《燕闲清赏笺•四时花纪》（明万历初年即1591年高濂撰）载：番椒“丛生白花，子俨秃笔头，味辣色红，甚可观”。章穆《调疾饮食辨》（成书于嘉庆十八年（1814年）载：“近数十年，群耆一物名辣枚又名辣椒。”辣椒传入我国经过历代劳动者辛勤培育，其品种之多跃居世界首位，如柿子椒、灯笼椒、羊角椒、佛手椒等，美人椒便是其中最优质的品种之一。
